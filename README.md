# Internship Tracker System

## Course

IMS566 Advanced Web Design Development and Content Management

## Project Description

Internship Tracker System is a PHP-based web application prototype developed for students to manage internship applications.

The system allows users to:

- login
- monitor internship applications
- manage internship preparation tasks
- review internship progress
- visualise progress using charts

## Features

- Authentication page
- Register page
- Dashboard
- Internship applications page
- Task tracking page
- Reports page
- Chart.js visualisation
- Responsive Bootstrap design
- Dark theme interface
